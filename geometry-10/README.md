# Геометрия 10 — Интенсив

Автономный интерактивный курс стереометрии за 10 класс. Отдельная папка, не связан с основной
системой, работает офлайн из браузера.

## Как пользоваться

1. Откройте **`index.html`** двойным кликом (откроется в браузере).
2. На главной — карта курса. Идите по порядку: Модуль 0 → 1 → 2 → 3 → 4 → 5.
3. Внутри урока: покрутите 3D-модель мышью, читайте теорию, разбирайте примеры, решайте задачи
   (кнопка «Проверить» даёт разбор).
4. В конце урока нажмите **«Отметить пройденным»** — прогресс сохранится и подсветится на карте.

Ничего устанавливать не нужно. Интернет не требуется: формулы (MathJax) и 3D-модели встроены.

## Что внутри

- `index.html` — карта курса и прогресс.
- `PLAN.md` — план интенсива на 1,5–2 месяца (расписание, контрольные точки).
- `assets/` — общий движок:
  - `styles.css` — оформление;
  - `app.js` — прогресс, формулы, навигация;
  - `figures.js` — 3D-движок фигур (вращение, сечения);
  - `quiz.js` — задачи с проверкой;
  - `course.js` — реестр уроков;
  - `vendor/tex-svg-full.js` — рендер формул (офлайн).
- `modules/` — уроки.

## Прогресс

Прогресс хранится локально в браузере (localStorage). Если открыть в другом браузере/на другом
компьютере — прогресс начнётся заново. Сбросить можно кнопкой на главной.

## Технически

- Чистый HTML/CSS/JS, без сборки и сервера.
- 3D — параллельное (ортографическое) проектирование, как в учебнике: невидимые рёбра штрихуются.
- Проверка ответов — на клиенте, без обращения к сети.
